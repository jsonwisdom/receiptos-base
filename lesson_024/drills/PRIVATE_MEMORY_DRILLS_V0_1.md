# Private Memory Drills v0.1

Mock-only cognitive rehearsal drills for Claimlandia Voice: Private Phone Edition.

## Drill 001: Life Lesson Capture

Prompt:
What happened, and what did it teach you?

Output:
Private local memory draft.

Blocked:
Diagnosis, legal conclusion, benefits prediction.

## Drill 002: War Story Sequence

Prompt:
What happened before, during, and after?

Output:
Timeline clarity note.

Blocked:
Credibility judgment, intent interpretation, service-connection conclusion.

## Drill 003: Stressful Moment Mapping

Prompt:
What was the moment like, and what did it affect?

Output:
Functional impact note.

Blocked:
PTSD conclusion, medical assessment, rating estimate.

## Drill 004: Attention Return

Prompt:
What is the next smallest step?

Output:
One-step continuation.

Blocked:
Cognitive impairment label, clinical interpretation.

## Drill 005: Drift Mirror

Prompt:
Read this back without judging me.

Output:
Possible drift note using safe language only.

Blocked:
Contradiction-as-proof, pattern-as-proof, credibility scoring.

## Drill 006: Receipt Lock

Prompt:
Lock this locally.

Output:
Local receipt with authority:false and cloud_upload:false.

Blocked:
Auto-upload, VA submission, cloud sync by default.

## Final State

Private phone rehearsal drills only.
