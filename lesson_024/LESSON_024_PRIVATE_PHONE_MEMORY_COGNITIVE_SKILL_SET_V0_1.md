## Cognitive Skill Set Enabled by Default

### Memory Capture

The system helps capture life lessons, war stories, stressful moments, and functional impact notes as private local drafts.

Allowed:
- user-stated memory
- user-stated sequence
- user-stated impact
- private local receipt

Blocked:
- diagnosis
- medical conclusion
- legal conclusion
- VA outcome claim

### Sequence Discipline

The system helps place memories into before, during, after, and impact fields.

Allowed:
- timeline organization
- missing-sequence prompt
- local clarity note

Blocked:
- credibility conclusion
- intent interpretation
- contradiction-as-proof

### Attention Support

The system uses one-step-at-a-time interaction.

Allowed:
- return to last step
- reduce screen load
- prompt for one next action
- resume unfinished draft

Blocked:
- cognitive impairment label
- medical assessment
- mental health conclusion

### Drift Mirror

The system flags possible story drift without judgment.

Allowed language:
- I found a possible story drift.
- This part may need clearer timing.
- This detail changed from the earlier note.
- You may want to review this before locking the receipt.
- Nothing here proves or disproves a claim.

Blocked language:
- You contradicted yourself.
- This damages credibility.
- This proves PTSD.
- This weakens your claim.
- This supports service connection.

### Functional Impact Mapping

The system turns user-stated experience into observable daily-life impact notes.

Allowed:
- daily task impact
- sleep disruption description
- social exposure difficulty
- work or routine interruption
- user-stated functional effect

Blocked:
- rating estimate
- eligibility claim
- benefits prediction
- service-connection conclusion

### Emotional Load Labeling

The system may help label intensity in plain language.

Allowed:
- stressful
- overwhelming
- difficult
- exhausting
- intense

Blocked:
- diagnosis
- disorder label
- medical causation
- psychiatric conclusion

### Safe Pointer Creation

The system creates mock or local pointers without uploading raw records.

Allowed:
- mock:// pointer
- local-only reference
- user-controlled export

Blocked:
- raw record upload
- automatic cloud sync
- hidden sharing
- real VA system submission

### Receipt Locking

The system locks a local receipt showing what was captured and what was not claimed.

Required receipt flags:
- authority:false
- fake_green:false
- cloud_upload:false
- diagnosis:false
- benefits_prediction:false
- va_outcome_prediction:false
