# Configuration Template

Set these values as GitHub Actions repository secrets or local environment variables before rendering.

```text
Replicate API credential: set as repository secret named REPLICATE_API_TOKEN
Replicate model: optional repository variable or secret named REPLICATE_MODEL
Google Drive root folder id: set as GDRIVE_ROOT_FOLDER_ID
Google Drive service account JSON: set as GDRIVE_SERVICE_ACCOUNT_JSON
Local credential path for manual runs: credentials.json
```
